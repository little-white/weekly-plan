# 每周计划
不知不觉一年又过去了，作为一个年长的前端开发，我时常惶恐。担心的不是与年轻同事的竞争，担心的不是各种眼花缭乱技术的学习，而是积累。

在这里开启我的一周计划，__目的__是：

* 打牢基础，巩固之前学到的知识
* 学习新的技能，通过不断的事件提升快速学习的能力
* 让我自己的技能树茂盛起来
* 文字的记录，让自己有所沉淀
